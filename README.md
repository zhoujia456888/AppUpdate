# AppUpdate
从项目中剥离出来的一个APP自动更新功能。主要实现APP的自动更新和后台下载。<br>
分强制更新和非强制更新，主要区别就在于dialog上有没有“取消”这一项。json文件可以直接复制url到地址查看。<br>
apk文件直接用的IT之家，可以在服务器json文件里将地址替换成自己的文件地址。
